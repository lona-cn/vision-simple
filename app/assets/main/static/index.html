Vision Simple - 跨平台视觉推理库

项目简介

vision-simple 是一个基于 C++23 的跨平台视觉推理库，提供开箱即用的推理功能。支持 YOLO 系列（YOLOv10 和 YOLOv11）、PaddleOCR 等视觉模型，并内建 HTTP API。通过 ONNXRuntime 支持多种推理平台，如 DirectML、CUDA、TensorRT 以及某些特殊设备（如 RockChip 的 NPU）。

功能特点

  • 支持 YOLOv10 和 YOLOv11 视觉推理模型
  • 内建 PaddleOCR 支持，进行图像文字识别
  • 基于 ONNXRuntime，支持 DirectML、CUDA、TensorRT 等多个执行平台
  • 内建 HTTP API，支持在线推理
  • 跨平台支持：Windows、Linux、WSL 等
